# 📊 Flipkart Mobile Sales Analysis

## 🔍 Overview

This project focuses on analyzing mobile phone sales data to extract
meaningful insights about pricing, discounts, customer ratings, and
brand performance. The goal is to apply data preprocessing, exploratory
data analysis (EDA), and visualization techniques to support data-driven
decision-making.

------------------------------------------------------------------------

## 🎯 Objectives

-   Clean and preprocess raw e-commerce data\
-   Perform exploratory data analysis (EDA)\
-   Understand relationships between price, discount, and ratings\
-   Identify top-performing brands and products\
-   Generate actionable business insights

------------------------------------------------------------------------

## 🛠️ Technologies Used

-   Python
-   Pandas & NumPy
-   Matplotlib & Seaborn
-   Jupyter Notebook

------------------------------------------------------------------------

## 📂 Dataset Features

-   Product Name\
-   Actual Price\
-   Discount Price\
-   Stars (Ratings)\
-   Number of Ratings\
-   Number of Reviews

------------------------------------------------------------------------

## ⚙️ Data Preprocessing

-   Removed currency symbols (₹) and commas from price columns\
-   Converted text data into numeric format\
-   Handled missing values (`NIL`, empty strings)\
-   Created feature: **Discount Percentage (%)**\
-   Extracted **Brand** from product name

------------------------------------------------------------------------

## 📊 Exploratory Data Analysis (EDA)

-   Brand performance analysis\
-   Pricing comparison (Actual vs Discount price)\
-   Discount impact on ratings\
-   Customer behavior insights

------------------------------------------------------------------------

## 📈 Visualizations

-   Bar Plot: Brand vs Ratings\
-   Scatter Plot: Price vs Discount Price\
-   Histogram: Rating Distribution\
-   Heatmap: Feature Correlation

------------------------------------------------------------------------

## 🔥 Key Insights

-   Top brands maintain higher ratings\
-   Budget phones (\< ₹17,000) perform competitively\
-   Discounts do not always improve ratings\
-   Strong correlation between price variables (expected)\
-   Slight negative correlation between discount % and ratings

------------------------------------------------------------------------

## 🚀 Conclusion

This project demonstrates how data analysis can provide meaningful
business insights from raw e-commerce data.
